# PV-Curve RAG Agent

Simple local RAG-based AI agent for Power-Voltage curve analysis using DeepSeek and vector embeddings.

## Installation & Run

Run LLM locally:

```bash
cd /server/ai
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
python main.py
```
